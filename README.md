# FlopToday

Een Open-Source SomToday alternatief, die je zelf kunt modificeren, hosten en zo veel meer.


## FAQ

#### Wanneer is FlopToday af?

Dat is onbekend en kan lang duren, we hebben levens en we werken er aan wanneer we kunnen.

#### Gaat FlopToday Floppen?

Hoe moet ik dat weten???

#### Werkt het precies als Somtoday?

Zeker, maar sommige functies kunnen afwijken tot de echte somtoday (tijdelijk of permanent), maar ons doel is om het zo vergelijkbaar mogelijk te maken met Somtoday.
## Security Vulnerabilities
